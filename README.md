# FSBO Support

Live site: https://juanfnmb1lpt.github.io/FSBO-SUPPORT/

FSBO Support is a React + Vite landing site with a support form flow for homeowners selling without an agent.

## Tech Stack

- React 18
- React Router
- Vite
- Tailwind CSS (CDN)

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Auto Deploy

This repo is configured to auto-deploy to GitHub Pages on every push to `main` via:

- `.github/workflows/deploy.yml`

In GitHub, make sure `Settings > Pages > Source` is set to `GitHub Actions`.
